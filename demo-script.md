## 슬라이드 4 — ChatGPT vs 클로드 코드

**[화면: 비교 테이블]**

> "ChatGPT 쓰시는 분 손 들어보세요.
>
> ChatGPT한테 '상세페이지 만들어줘' 하면 뭐가 나올까요?
> 텍스트가 나옵니다. 글자요.
> 그거 복사해서 어딘가에 붙여넣고, 또 수정하고... 번거롭죠.
>
> 클로드 코드는 다릅니다.
> '상세페이지 만들어줘' 하면 진짜 파일이 만들어집니다.
> 브라우저에서 바로 열어볼 수 있는 상세페이지가 뚝딱.
>
> 표를 보시면요.
> 결과물 — ChatGPT는 텍스트, 클로드 코드는 실제 파일.
> 수정 — ChatGPT는 다시 물어봐야 하고, 클로드 코드는 '바꿔줘' 한마디.
> 기억력 — ChatGPT는 매번 처음부터, 클로드 코드는 업무 매뉴얼을 읽고 일합니다.
>
> 비유하면요.
> ChatGPT는 프리랜서한테 카톡하는 거고,
> 클로드 코드는 사무실에 직원 앉혀놓는 겁니다."

## 슬라이드 14 — 전체 흐름

> "전체 흐름을 먼저 보여드릴게요. 딱 4단계입니다.

> 빈 폴더에 업무 매뉴얼 하나 넣고, '초기화해줘' 하면 세팅 끝.
> 그 다음부터 말로 시키면 됩니다."

## 슬라이드 36 — 발표자 전용 (비상 대응)

**[화면: 비상 대응 + 시간 조절 — 청중에게 안 보여줌]**

### 비상 대응

| 상황 | 대응 |
|------|------|
| 인터넷 끊김 | 모바일 핫스팟 전환 |
| 에러 발생 | "에러도 알아서 고칩니다" (장점으로 전환) |
| 결과물 안 나옴 | 미리 만들어둔 백업 HTML 보여주기 |
| 인터뷰 모드 안 작동 | 직접 상품 정보 입력하고 진행 |
| 컨텍스트 부족 | /clear 후 재시작 |
| 크레딧 소진 | 백업 결과물로 전환 |
| 화면 안 보인다고 할 때 | 글씨 크기 키우기 (Cmd+= 여러 번) |

### 시간 조절

| 상황 | 조절 |
|------|------|
| 시간 부족 (-15분) | 이미지 변환 스킵, 보너스 하나만 |
| 시간 부족 (-30분) | PART 1~2를 10분으로 줄이고 바로 시연 |
| 시간 여유 (+15분) | CS 답변 라이브 시연 추가 |
| 시간 여유 (+30분) | 마케팅 코치 시연 + 참석자 실습 |
| 질문 많음 | 보너스 줄이고 Q&A 늘리기 |

### 강연 중 반복할 핵심 메시지 (수시로)

> "코드 안 칩니다."
> "결과만 보면 됩니다."
> "말하면 됩니다."
> "몰라도 됩니다. AI가 알아서 합니다."
> "마음에 안 들면 바꿔달라고 하면 됩니다."
